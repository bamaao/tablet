# 入库功能使用说明

## 支持的入库模式

### 模式一：散装入库
**场景**：零散的药品，不按包装计算

**操作步骤**：
1. 选择"散装入库"
2. 输入数量（如：500）
3. 单位自动显示为"g"（克）
4. 点击确认

**示例**：
```
散装入库 500g
→ 散装库存增加 500g
```

---

### 模式二：包装入库（可自定义包装规格）
**场景**：按包装入库，每包规格可自由设置

#### 示例1：500克/包
```
入库类型：包装入库
入库数量：10（包）
包装规格：500（克/包）← 手动输入或点击快捷按钮

系统计算：
  10包 × 500克/包 = 5000克

结果：
  包装库存增加 10包（500g规格）
  总库存增加 5000g
```

#### 示例2：200克/包
```
入库类型：包装入库
入库数量：5（包）
包装规格：200（克/包）

系统计算：
  5包 × 200克/包 = 1000克

结果：
  包装库存增加 5包（200g规格）
  总库存增加 1000g
```

#### 示例3：250克/包
```
入库类型：包装入库
入库数量：8（包）
包装规格：250（克/包）

系统计算：
  8包 × 250克/包 = 2000克

结果：
  包装库存增加 8包（250g规格）
  总库存增加 2000g
```

---

## 包装规格快捷按钮

当选择"包装入库"时，系统提供以下快捷选项：

| 按钮 | 功能 |
|------|------|
| **默认** | 使用药品的默认包装规格 |
| **250g** | 快速设置为250克/包 |
| **500g** | 快速设置为500克/包 |

**也可以手动输入任意规格**，如：
- 100克/包
- 150克/包
- 300克/包
- 1000克/包
- 等等...

---

## 实时计算显示

当您输入包装入库的数量和规格时，系统会实时显示：

```
共计: 10 包 × 500 g/包 = 5000 g
```

让您清楚知道将入库多少总重量。

---

## 多规格包装库存

系统支持**同时存储多种规格的包装**：

### 实际案例
```
第一次入库：5包 × 500g/包 = 2500g
第二次入库：3包 × 250g/包 = 750g
第三次入库：2包 × 200g/包 = 400g

最终库存显示：
  • 500g/包 × 5包
  • 250g/包 × 3包
  • 200g/包 × 2包
  总库存：3650g
```

---

## 出库时的智能匹配

当您使用**智能出库模式**时，系统会自动计算最优组合：

### 示例
```
当前库存：
  • 500g/包 × 2包
  • 250g/包 × 3包
  • 200g/包 × 2包
  • 散装：100g

出库需求：700g

智能方案：
  ✅ 1包 500g/包
  ✅ 1包 200g/包
  总计：700g（无需拆包！）
```

---

## 总结

✅ **散装入库**：直接以克为单位入库
✅ **包装入库**：以包为单位，可自定义每包规格
✅ **多规格共存**：同一种药可有多种不同规格的包装
✅ **智能计算**：系统自动处理包装与克重的换算
✅ **灵活快捷**：提供常用规格快捷按钮，支持手动输入
