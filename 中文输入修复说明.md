# 中文输入修复说明

## 问题描述
在React Native应用中，某些TextInput组件无法输入中文。

## 修复方案
为所有需要支持中文输入的TextInput组件添加以下属性：
```typescript
keyboardType="default"     // 使用默认键盘（不是数字键盘）
autoCorrect={true}         // 启用自动纠正（帮助中文输入）
autoCapitalize="sentences" // 首字母大写（对于中文无影响，但有助于输入法）
```

## 数字输入框（未修改）

以下输入框保持为数字键盘，不需要中文输入：
- 入库数量（decimal-pad）
- 包装规格（number-pad）
- 出库数量（decimal-pad）
- 拆包数量（decimal-pad）
- 药品表单中的每包含量（decimal-pad）
- 药品表单中的最低库存（decimal-pad）
- 处方付数（number-pad）

## 技术说明

### keyboardType 的取值
- `"default"` - 默认键盘，支持中文输入 ✅
- `"decimal-pad"` - 数字+小数点，不支持中文
- `"number-pad"` - 纯数字，不支持中文

### autoCorrect 的取值
- `true` - 启用自动纠正，有助于中文输入法的联想功能 ✅
- `false` - 禁用自动纠正，会影响中文输入体验

### autoCapitalize 的取值
- `"none"` - 不自动大写，适合拼音输入
- `"sentences"` - 句首大写，适合中文文本输入 ✅
- `"words"` - 单词首字母大写
- `"characters"` - 全部字符大写
